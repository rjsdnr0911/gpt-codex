# Codex Craft

브라우저에서 실행되는 한국어 싱글플레이 3D 블록 생존 게임입니다. Vite, TypeScript, Three.js로 만들었고, 절차적 지형, 동굴, 광물, 제작, 제련, 횃불 조명, 상자 보상, 첫 적대 몹을 지원합니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173/`을 엽니다.

## 스크립트

```bash
npm test
npm run build
```

## 조작

- `WASD`: 이동
- `마우스`: 시점 이동
- `좌클릭`: 채굴 또는 공격
- `우클릭`: 설치, 사용, 상자 열기
- `E`: 인벤토리
- `Esc`: 일시정지 또는 창 닫기
- `1-9`: 핫바 선택
- `Ctrl`: 달리기
- `R`: 브라우저용 달리기 보조키
- `Shift`: 웅크리기
