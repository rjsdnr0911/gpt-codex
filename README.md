# Voxel Frontier

A browser-based singleplayer voxel survival game built with Vite, TypeScript, and Three.js.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5173/`.

## Scripts

```bash
npm test
npm run build
```

## Controls

- `WASD`: Move
- `Mouse`: Look
- `Left click`: Mine
- `Right click`: Place/use
- `E`: Inventory
- `Esc`: Pause/close menu
- `1-9`: Hotbar
- `Ctrl`: Sprint
- `R`: Sprint fallback for browsers
- `Shift`: Sneak
